# Machine-Learning-Project
Sustainability Investment Profile: Using Machine Learning Models for ESG Firm categorising 

Developed an end-to-end machine learning pipeline to classify 2,000 corporations into ESG investment categories (Loser, Normal, Winner, Champion) using a 16-feature dataset of environmental, social, and governance metrics. Conducted rigorous exploratory data analysis including outlier detection, missing value imputation, and multicollinearity assessment using both Pearson correlation and Cramér's V, before engineering and standardising features for modelling. Built and evaluated four classification models — Multinomial Logistic Regression, Elastic Net Logistic Regression, Gradient Boosting, and Random Forest — assessing each against accuracy, ROC-AUC, confusion matrices, sensitivity analysis, and 10-fold cross-validation. Applied PCA-based decision boundary visualisation to compare linear and non-linear classification behaviour under class imbalance. Recommended Logistic Regression for deployment on the basis of its lowest economically harmful misclassification rate, regulatory explainability alignment with MiFID II RTS 6 and SFDR, and consistent generalisation to a held-out test set of 1,000 firms. Framed all analytical conclusions in terms of business risk — distinguishing greenwashing risk from reputational risk — and produced a forward-looking strategic assessment of how LLM-based controversy monitoring and satellite-derived environmental data could enhance ESG signal quality in a live trading environment.
